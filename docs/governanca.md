# Regras de Governança do Repositório
## O que pode ser versionado
- READMEs
- Scripts de exemplo
- Prints sem dados sensíveis
- Diagramas
- Arquivos de documentação
- Fluxos exportados sem credenciais
## O que não pode ser versionado
- Senhas
- Tokens
- Chaves de API
- Service role do Supabase
- Arquivos .env reais
- Dados pessoais de clientes ou pacientes
