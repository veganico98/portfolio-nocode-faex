# Portfólio No-Code — FAEX
## Sobre o repositório
Este repositório reúne as atividades desenvolvidas na disciplina No-Code I e II da
Pós-Graduação em Desenvolvimento Full Stack da FAEX.
## Objetivo
Organizar, documentar e versionar os trabalhos práticos realizados ao longo da
disciplina, incluindo automações, integrações, APIs, webhooks, n8n, Supabase,
governança, segurança e código customizado.
## Estrutura
- Semana 01 — Fundamentos de automação e No-Code
- Semana 02 — APIs, JSON e Postman
- Semana 03 — Webhooks e eventos
- Semana 04 — Lógica visual e regras de negócio
- Semana 05 — Resiliência e tratamento de erros
- Semana 06 — Código customizado com JavaScript
- Semana 07 — Banco de dados com Supabase
- Semana 08 — Governança, autenticação e segurança
## Ferramentas utilizadas
- n8n
- Tally
- Supabase
- Postman
- VS Code
- GitHub
- JavaScript
